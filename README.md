# Project Overview

This portfolio website showcases various projects and skills.

# Tech Stack
- HTML
- CSS
- JavaScript
- React

# Setup Instructions
1. Clone the repository: `git clone https://github.com/aborgesl-cloud/portfolio-website.git`
2. Navigate to the project directory: `cd portfolio-website`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

# Features
- Responsive design
- Interactive UI components
- Easy to navigate the project showcase.
